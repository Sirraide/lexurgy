Lexurgy
=======

Lexurgy is a sound change applier. It allows the user to simulate historical changes
in spoken languages by applying regular rules to a list of words.

Found a bug? Want to request a feature? Open an issue on `GitHub <https://github.com/def-gthill/lexurgy/issues>`_.
